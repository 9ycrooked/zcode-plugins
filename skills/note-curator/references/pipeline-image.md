# 管线：图片导入

本文件在 SKILL.md 路由到图片导入时按需加载。

## 工作流

1. **上传**：使用 `file.upload_asset` 将图片上传到思源 assets 目录
2. **OCR**（可选）：使用 `file.get_image_ocr_text` 提取图片中的文字
3. **分析**：描述图片内容和知识要点（如无法 OCR）
4. **写入文档**：
   ```
   # [从图片中提取的核心命题]

   ![图片说明](assets/上传后的路径.png)

   [从图片中识别的知识要点，用自己的话表述]

   > OCR 识别原文（如有）：[原文内容]

   ---

   - 来源：[图片来源 / 截图日期]
   - 类型：[概念 / 图示 / 截图 / 思维导图]
   - 关联：[[相关知识文档]]
   - 标签：#topic
   ```
5. **去重 + 建链 + 归档**：同纯文本管线

## 支持的图片格式

PNG / JPG / JPEG / GIF / WEBP / SVG

## 注意事项

- 截图、思维导图、手写笔记均可 OCR，但准确率依赖图片质量
- 如果图片信息量很大（如长文截图），优先用 OCR 提取文字再原子化拆分
- 多张相关图片可合并为一条知识文档，每张图单独上传并嵌入
