# Zettelkasten 实践指南

本文件供 skill 在需要时参考，不直接呈现给用户。

## 核心原则

1. **原子性**：一条笔记 = 一个想法。如果一段话里有两个独立想法，拆成两条
2. **自主性**：用自己的话写，不照搬原文。能用自己的话解释，才算真正理解
3. **链接性**：每条笔记至少链接到一条已有笔记。知识的价值在关联，不在孤立
4. **可检索性**：标题写命题句，不要写日期或模糊标题

## 标题写法示例

| ❌ 差 | ✅ 好 |
|-------|-------|
| 读书笔记 2024.1.1 | 费曼学习法的本质是输出倒逼输入 |
| AI 相关 | Transformer 的自注意力机制解决了长距离依赖问题 |
| 第三章笔记 | 心流状态需要挑战与技能的动态平衡 |
| 待整理 | DRY 原则的局限：过度抽象反而增加认知负担 |

## 链接策略

- **向上链接**：这条知识属于哪个更大的概念？（→ 更抽象的知识文档）
- **向下链接**：这个概念包含哪些子概念？（→ 更具体的知识文档）
- **横向链接**：这个概念和什么相似/对立/互补？（→ 平行知识文档）
- **来源链接**：这个知识来自哪里？（→ 原始材料）

## 什么时候记

不需要把所有东西都原子化。以下情况值得记：
- 产生了"原来如此"的顿悟
- 发现了两个看似无关事物的联系
- 纠正了一个长期以来的错误认知
- 学到了一个可复用的方法/模式
