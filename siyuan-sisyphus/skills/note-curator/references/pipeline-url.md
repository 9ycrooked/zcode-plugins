# 管线：URL / 网页导入

本文件在 SKILL.md 路由到 URL 导入时按需加载。

## 工作流

1. **抓取**：使用 `WebFetch` 工具抓取网页内容
   - 参数：`url` = 用户提供的 URL, `prompt` = "提取正文内容并转为 markdown"
2. **提取**：从返回的 Markdown 中提取正文，过滤导航栏/广告/侧边栏
3. **评估长度**：
   - 短文（< 2000 字）：整体提炼为一条原子笔记
   - 长文（> 2000 字）：按章节/段落拆分为多条原子笔记，创建索引页
4. **写入文档**：
   ```
   # [从文章中提取的核心命题]

   [用自己的话重述知识要点]

   > 原文参考：[关键引用段落]

   ---

   - 来源：[文章标题](https://example.com/article)
   - 类型：[概念 / 方法 / 案例 / 新闻 / 教程]
   - 关联：[[相关知识文档]]
   - 标签：#topic
   ```
5. **处理图片**：如果网页包含关键图片，下载后用 `file.upload_asset` 上传并嵌入
6. **去重 + 建链 + 归档**：同纯文本管线

## 注意事项

- 如果网页需要登录或被反爬，告知用户改为手动复制文本导入
- 抓取失败时回退方案：请用户手动粘贴正文内容
