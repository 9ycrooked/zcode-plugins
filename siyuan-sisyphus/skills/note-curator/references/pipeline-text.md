# 管线：纯文本导入

本文件在 SKILL.md 路由到文本导入时按需加载。

## 工作流

1. **提取**：从输入中识别核心知识点，剥离冗余上下文
2. **原子化**：如果内容包含多个独立知识点，拆分为多条原子笔记
3. **写入**：为每条知识点创建一个文档，结构：
   ```
   # [核心命题]

   [正文：用自己的话重新表述，2-5 段]

   ---

   - 来源：[书名 / 对话日期 / 上下文说明]
   - 类型：[概念 / 方法 / 案例 / 引用 / 术语]
   - 关联：[[已有知识文档]]
   - 标签：#topic1 #topic2
   ```
4. **去重**：用 `search.fulltext` 搜索已有知识库，避免重复创建
5. **建链**：找到相关文档，添加 `[[双链]]`
6. **归档**：放入对应笔记本（Projects / Areas / Resources）

## 关键原则

- 不照搬原文，必须用自己的话重述
- 标题写命题句："费曼学习法的本质是输出倒逼输入"，不要写"读书笔记20240101"
- 如果用户原文已经很精炼，可以在保留原文的基础上补充自己的理解
