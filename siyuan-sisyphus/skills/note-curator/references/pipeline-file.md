# 管线：PDF / DOCX 文件导入

本文件在 SKILL.md 路由到文件导入时按需加载。

## PDF 导入

1. **提取**：使用 `pdf` skill 提取 PDF 文字内容
2. **评估长度**：
   - 短文档（< 10 页）：整体提炼为一条原子笔记
   - 长文档（> 10 页）：按章节拆分为多条原子笔记
3. **写入文档**：
   ```
   # [从文档中提取的核心命题]

   [用自己的话重述知识要点]

   > 原文参考：[关键引用段落]

   ---

   - 来源：[文档标题] (本地路径或文件名)
   - 类型：[概念 / 方法 / 案例 / 论文 / 报告]
   - 关联：[[相关知识文档]]
   - 标签：#topic
   ```
4. **去重 + 建链 + 归档**：同纯文本管线

## DOCX 导入

1. **提取**：使用 `docx` skill 提取 Word 文档内容
2. **后续流程**：同 PDF 导入

## 注意事项

- 扫描版 PDF（纯图片）无法直接提取文字，需提示用户先 OCR 或手动复制
- 表格密集的文档，提取后可用思源表格重新排版
- 如果文件较大，告知用户预计拆分的笔记数量，获确认后再批量创建
